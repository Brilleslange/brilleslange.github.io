Color assignment

About

If you're like me, when hosting Twilight Imperium games, you like assigning colors to players based on the faction they've chosen. Clearly, if someone plays Arborec, they have to be green! But what if you're playing a game with the Sardakk N'orr, the Nekro Virus, the Barony of Letnev, the Embers of Muaat, and the L1Z1X Mindnet, all of whom have red/black tokens? Well, this website is for you!

Twilight Imperium 4th Edition Color Assigner (TICA) assigns colors to players based on which colors best match their factions. Each faction has a score for each color, and the program attempts to maximize that score to find the best possible color combination. If you disagree with the scores I've assigned, don't worry, you can enter custom color scores!

TICA was first created as a Python application in 2019 and was rewritten into a web application with Javascript in 2022, with support for the Prophecy of Kings expansion and additional functionality for excluding colors and avoiding color combinations.

Options

Expansions

If you have an expansion, you can check the box for the expansion here. Content from unchecked expansions is hidden. All expansions are checked by default.

Speaker token

Check the box if you want to assign the speaker token to a random player.

Exclude colors

If there are any colors you would like to exclude - for example, if you find them hard to see on the board - you can check their box, and they will be excluded from the color assignment.

Avoid color combos

If there are any combinations of colors you would like to avoid - for example, if a player is colorblind - you can check their box, and the they will be avoidded during color assignment. For example, if you check the box for Red and Green, then red or green may be assigned, but never both at the same time.

Functionality

Faction randomizer

Enter a number and press the "Randomize" button. That number of factions will be randomly selected.

Color assigner

Pressing "Assign colors" will assign colors to the selected factions based on which colors match them best. Due to the way the algorithm is written, pressing the button multiple times may give different results. See below for details on how the color assigner works.

Scores

Scores have been assigned based on these criteria (with some exceptions):

  • 10 points: The only color on the control token's faction symbol side.
  • 9 points: The background color on the control token's faction symbol side.
  • 8 points: The color appears on the faction symbol.
  • 7 points: The color of the ship on the control token's fleet symbol side.
  • 5 points: A main color on the faction sheet.
  • 4 points: A secondary color on the faction sheet.
  • 2 points: An accent color on the faction sheet.
  • 1 points: An complimentary color to the faction sheet.
Scores can be edited. They will revert to default scores if the page is refreshes, but you may export your custom scores and import them later.

To export scores, press the "Export" button and copy and paste the text from the popup into a text editor such as Notepad on your desktop or the Notes app on your phone. Save the file somewhere for future use.

To import scores, press the "Import" button and copy and paste the previously saved text string into the popup and press "OK".
